# Project Overview

**App Name:** Reema's Action Tracker (Machli)
**URL:** https://reema-machli-app.vercel.app
**Repo:** https://github.com/gauravvermatechno/reema-machli-app
**Type:** Progressive Web App (PWA)
**Created:** 2026-04-13

## Purpose
A beautiful, dopamine-releasing task tracker built for Reema to manage work actions across team members. Features inspirational quotes from women athletes, poets, leaders, and scientists - including 140 Bengali quotes in native script.

## Key Features
- **Login** - Hardcoded username `reema` (case-insensitive)
- **Dashboard** - Progress ring, stats cards, upcoming deadlines, rotating quotes
- **Task Management** - Full CRUD with subtasks, hours tracking, sorting, filtering
- **997 Inspirational Quotes** - From women across 10 categories
- **140 Bengali Quotes** - In proper Bengali script, shown 1-in-3 rotations
- **Swimming Fish** - Interactive mascot (click for bubbles + hearts!)
- **PWA** - Installable on phones and desktops, works offline
- **Cloud Persistence** - Vercel Blob storage survives redeployments
